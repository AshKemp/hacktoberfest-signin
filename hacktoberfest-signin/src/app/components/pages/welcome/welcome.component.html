Welcome to Hacktoberfest 2020

Every year Digital Ocean hosts an awesome Hackathon for Github users to learn to collaborate and contribute to open source projects

In the spirit of this I have created a first time project for new and old users to help them get their first PR of the Season

This project features Github users and their prefered languages in a visual graph, this will program will ask for your Github username and your programming language preferences

In order to participate in hacktoberfest you will need to make 4 Pull Requests in opensource projects throughout Github.

In order to make a vaild pull request in this project follow the documentation inside the README of this repo

This project can help with your goals this year by adding yourself to the list of users who have added their language preferences

Happy Hacking!
