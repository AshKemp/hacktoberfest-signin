Welcome to Hacktoberfest 2020

Every year Digital Ocean hosts an awesome Hackathon for Github users to learn to collaborate and contribute to open source projects

In the spirit of this I have created a first time project for new and old users to help them get their first PR of the Season

This project features Github users and their prefered languages in a visual graph

In order to participate on this project and in hacktoberfest you will need to make a PR in this project and add yourself to the list
